# 安装

# 环境需要

要安装 Torchjs, 你需要首先安装 Node.js (opens new window)。 国内用户可以安装 cnpm (opens new window) 加快 NPM 模块安装速度。

# 安装

$ npm i torchjs -g

# 示例

$ git clone git@github.com:antvis/g2.git
$ cd g2
$ npm i
$ npm run test-live
